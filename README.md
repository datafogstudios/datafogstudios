# DataFog Studios™

**Free Premium Programming Education Platform**

Master the top 10 programming languages with interactive lessons, real-world projects, and professional certifications — **100% free forever**.

## Live Site
https://datafogstudios.netlify.app

## Features
- Advanced course catalog with real-time search & multi-filtering
- Personal learning dashboard with progress tracking
- Wishlist system (persisted across sessions)
- Fully responsive design (mobile-first)
- Smooth navigation and modern UI

## Tech Stack
- HTML5, CSS3 (custom, no frameworks)
- Vanilla JavaScript (ES6+)
- Mobile menu, header effects, localStorage persistence

## Local Development
1. Clone the repo
2. Open `index.html` in your browser
3. All functionality works offline
