Control Flow: Conditional Statements

Make your programs smart by executing code based on conditions using if, elif, and else

Learning Objectives

  • Use if, elif, and else statements
  • Understand comparison and logical operators
  • Write nested conditionals
  • Handle user input with conditions

The if Statement

The if statement executes code only if a condition is True.

age = 18\n\nif age >= 18:\n print("You are an adult!")

if-elif-else Chain

Use elif for multiple conditions and else for the default case.

score = 85\n\nif score >= 90:\n grade = "A"\nelif score >= 80:\n grade = "B"\nelif score >= 70:\n grade = "C"\nelse:\n grade = "F"\n\nprint(f"Your grade: {grade}")

Comparison & Logical Operators

Operator Meaning Example
== Equal to x == y
!= Not equal x != y
)> , < , >= , <= Greater/Less than x > 10
and, or, not Logical operators age > 18 and is_student

Nested Conditionals

You can place if statements inside other if statements.

temperature = 25\nis_raining = False\n\nif temperature > 20:\n if is_raining:\n print("Stay indoors")\n else:\n print("Perfect day for a walk!")
