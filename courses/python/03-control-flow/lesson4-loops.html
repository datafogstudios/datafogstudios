Loops in Python

Repeat code efficiently using for and while loops — essential for automation and data processing

Learning Objectives

  • Use for loops with range() and iterables
  • Use while loops for condition-based repetition
  • Control loops with break, continue, and else
  • Understand nested loops and common patterns

The for Loop

The for loop iterates over a sequence (list, string, range, etc.).

# Basic for loop with range for i in range(5): print(f"Count: {i}") # Loop over a list fruits = ["apple", "banana", "orange"] for fruit in fruits: print(fruit)

Visualizing a for Loop

The while Loop

The while loop repeats as long as a condition is True.

count = 0 while count < 5: print(f"Count: {count}") count += 1

Loop Control Statements

  • break — exit the loop immediately
  • continue — skip to the next iteration
  • else — executes if loop completes normally (no break)

Nested Loops

Loops inside loops — useful for grids, matrices, etc.

for i in range(3): for j in range(3): print(f"({i}, {j})")
