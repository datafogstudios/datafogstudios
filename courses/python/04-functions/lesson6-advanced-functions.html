Advanced Functions

Take your functions to the next level with lambda, recursion, closures, and decorators

Learning Objectives

  • Use lambda functions for short, anonymous operations
  • Understand recursion and when to use it
  • Create higher-order functions and closures
  • Apply decorators to modify function behavior

Lambda Functions

Anonymous functions for simple operations — often used with map(), filter(), and sort().

# Regular function def square(x): return x ** 2 # Lambda equivalent square_lambda = lambda x: x ** 2 numbers = [1, 2, 3, 4, 5] squared = list(map(lambda x: x ** 2, numbers)) print(squared)

Recursion

A function that calls itself — powerful for problems with natural recursive structure.

def factorial(n): if n == 0: return 1 else: return n * factorial(n - 1) print(factorial(5)) # 120

Higher-Order Functions & Closures

Functions that take or return other functions.

def make_multiplier(factor): def multiplier(x): return x * factor return multiplier double = make_multiplier(2) print(double(5)) # 10

Decorators

Functions that modify other functions — used for logging, timing, access control.

def timer(func): import time def wrapper(*args, **kwargs): start = time.time() result = func(*args, **kwargs) end = time.time() print(f"{func.__name__} took {end - start:.4f} seconds") return result return wrapper @timer def slow_function(n): sum(range(n)) slow_function(1000000)
