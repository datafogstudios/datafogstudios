Variables & Data Types

Learn how to store and work with data in Python using variables and built-in types

Learning Objectives

  • Create and name variables following Python conventions
  • Understand common data types: int, float, str, bool
  • Perform type conversion
  • Use f-strings for formatted output

What is a Variable?

A variable is a named container for storing data. In Python, you create a variable by assigning a value using the = operator.

name = "Alice"
age = 25
height = 1.65
is_student = True

print(name)
print(age)
print(height)
print(is_student)

Variable Naming Rules

  • Must start with a letter or underscore
  • Can contain letters, numbers, underscores
  • Case-sensitive (Age ≠ age)
  • Use snake_case for readability (recommended)

Common Data Types

Type Example Description
int 42, -10 Whole numbers
float 3.14, -0.5 Decimal numbers
str "hello", 'Python' Text (strings)
bool True, False Boolean values

Type Conversion

Use built-in functions to convert between types:

number_str = "123"
number = int(number_str) # String to int
float_num = float("3.14") # String to float
text = str(100) # Int to string
boolean = bool(0) # 0 = False, anything else = True
